Node

2014: Notifier is a notification service. Messages are sent by scripts and applications over HTTP whenever something interesting happens. Delivery occurs over websockets, either to client scripts or a web browser UI. Built with Node and Angular, and deployed at OpenShift.

2014: Headlines is an RSS reader fills the void left by Google Reader. Built with Node, Angular, and Redis.

Python

2014: Medley is a hodge-podge of small utilities grouped under a single application. Built with Python3 and CherryPy for deployment to a Raspberry Pi.

PHP

2012: Phing recipes is a collection of deployment tasks for Phing, the Ant-based PHP build tool. Concatentation and minification, rsync-with-a-bit-of-rearrangement, and WordPress database migration take center stage.

  • Using Sphinx to provide search capabilities
  • Using Beanstalk task queues to schedule future work
